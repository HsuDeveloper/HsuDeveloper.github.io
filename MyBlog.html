Jekyll 없이 github blog 생성

Web에 대한 이해도와 디자인적인 부분 역시 직접 만들어 보며 실습을 위해 Jekyll없이 github blog를 만들어 보려고 합니다.

기본적으로 header, body(left_body,right_body), footer로 layout을 만들고 left_body에는 프로필과 메뉴를, right_body에는 home그리고 menu에 따른 내용을 볼 수 있게 만들었습니다.

직접 코딩하여 만들고 있다는 느낌을 주기 위해 전체 body의 background_image에 해당 blog의 css코드를 넣어봤습니다.

Code

html, css, javascript, jquery가 사용되었고, 계속 업데이트 할 예정입니다.

  • menu의 경우 javascript code와 css코드를 w3school에서 가져왔습니다. javascript언어를 통해 menu창의 검색기능을 추가하여 메뉴 검색시 편의성을 더하고, css언어를 통해 mouse hover가 발생할 때 색이 변하여 어떤 메뉴에 마우스가 올라 가 있는지 쉽게 확인 할 수 있습니다.

    • JavaScript Code

      function search_menu() {
            var input, filter, ul, li, a, i;
            input = document.getElementById("search_text");
            filter = input.value.toUpperCase();
            ul = document.getElementById("menu_ul");
            li = ul.getElementsByTagName("li");
            for (i = 0; i < li.length; i++) {
             a = li[i].getElementsByTagName("a")[0];
             if (a.innerHTML.toUpperCase().indexOf(filter) > -1) {
               li[i].style.display = "";
             } else {
               li[i].style.display = "none";
             }
            }
            }
